# TIGERHASH

Реализация криптографической хеш-функции `Tiger` на Python.

## Что делает алгоритм

Алгоритм принимает входные данные произвольной длины и преобразует их в фиксированный 192-битный хеш. Одинаковый ввод всегда даёт одинаковый результат, а даже небольшое изменение строки меняет хеш полностью.

## Как работает реализация

1. Входные данные дополняются по правилам Tiger:
   добавляется байт `0x01`, затем нули, после чего в конец записывается длина сообщения в битах.
2. Подготовленное сообщение разбивается на блоки по 64 байта.
3. Каждый блок проходит три раунда обработки с коэффициентами `5`, `7` и `9`.
4. Внутри раундов используются:
   операции `XOR`, 64-битные сложение, вычитание и умножение, а также S-box таблицы `T1`, `T2`, `T3`, `T4`.
5. После обработки блока внутреннее состояние обновляется, и в конце формируется итоговый 192-битный хеш.

## Особенности проекта

- Реализация основных шагов Tiger вручную на Python.
- Использование 64-битной арифметики с маской `0xFFFFFFFFFFFFFFFF`.
- Встроенная самопроверка `self_test()` на известных тестовых векторах.
